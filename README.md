# 🖋️ The Master Printmaker - Artisan Printing Studio

![Banner](https://images.unsplash.com/photo-1516962215378-7fa2e137ae93?auto=format&fit=crop&q=80&w=1200)

> **"Heritage in Every Impression. Precision for the Modern Age."**

Selamat datang di repositori resmi **The Master Printmaker**. Ini adalah platform landing page modern yang dirancang untuk atelier percetakan kelas atas, menggabungkan teknik cetak tradisional yang teliti dengan keunggulan digital masa kini.

---

## ✨ Fitur Utama

- 🎨 **Artisan Aesthetic**: Desain mewah dengan palet warna *Antique Gold* dan *Midnight Black*.
- 📱 **Fully Responsive**: Pengalaman mulus di Desktop, Tablet, hingga Smartphone.
- ⚡ **React & Vite**: Performa super cepat dengan arsitektur komponen yang modular.
- 💅 **Tailwind CSS v4**: Styling masa depan dengan *design tokens* yang presisi.
- 🛤️ **Modern Routing**: Navigasi instan tanpa reload antar halaman (Home, Portfolio, Process, About, Contact).
- 🖼️ **Interactive Gallery**: Portofolio dengan efek filter dinamis.
- 📥 **Project Inquiry**: Formulir konsultasi lengkap dengan fitur simulasi unggah berkas.

---

## 🛠️ Teknologi yang Digunakan

- **Frontend Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Typography**: Noto Serif & Manrope (via Google Fonts)
- **Icons**: Material Symbols Outlined

## 🚀 Cara Menjalankan Secara Lokal

Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/).

1. **Clone Repositori**
   ```bash
   git clone https://github.com/Muhfaizr21/Jasa-Printing.git
   cd Jasa-Printing
   ```

2. **Masuk ke Folder Frontend**
   ```bash
   cd frontend
   ```

3. **Instal Dependensi**
   ```bash
   npm install
   ```

4. **Jalankan Development Server**
   ```bash
   npm run dev
   ```

5. **Buka di Browser**
   Kunjungi `http://localhost:5173`
